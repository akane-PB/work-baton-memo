# Google Sheets共有設定

このアプリの画面はGitHub Pagesで開き、案件データは自分のGoogle Sheetsに保存します。

## 1. Google Sheetsを作る

1. Google Driveで新しいスプレッドシートを作る
2. 名前を `作業バトンメモ データ` などにする
3. メニューから `拡張機能` → `Apps Script` を開く

## 2. Apps Scriptを貼る

1. このフォルダの `google-apps-script.gs` の中身をコピーする
2. Apps Scriptの `コード.gs` に貼り付ける
3. `SHARED_SECRET` の文字を、自分だけがわかる長めの共有パスワードに変更する
4. 保存する

## 3. Webアプリとして公開する

1. Apps Script右上の `デプロイ` → `新しいデプロイ`
2. 種類は `ウェブアプリ`
3. `次のユーザーとして実行` は `自分`
4. `アクセスできるユーザー` は `全員`
5. デプロイする
6. 表示された `ウェブアプリURL` をコピーする

`アクセスできるユーザー` は `全員` ですが、共有パスワードがないと読み書きできない作りです。

## 4. アプリに設定する

1. GitHub Pagesのアプリを開く
2. `設定` を押す
3. `Apps Script URL` にウェブアプリURLを貼る
4. `共有パスワード` にApps Scriptで設定した文字を入れる
5. `設定を保存`

## 5. 使い方

- 端末の内容をGoogle Sheetsへ送る: `共有保存`
- Google Sheetsの内容を端末へ読む: `共有読込`

最初は、最新版データが入っている端末で `共有保存` してください。
その後、ほかの端末で `共有読込` すると同じ案件データを見られます。
